# AGV ROS2 Jazzy Project

## Requirements

- Ubuntu 24.04
- ROS2 Jazzy

## Installation

```bash
git clone <repository_url>
cd AGV_Project

chmod +x install.sh
chmod +x setup.sh
chmod +x run.sh

./install.sh
./setup.sh
```

## Run

```bash
./run.sh
```

## Project structure

```
firmware/
    ESP32 Arduino code

ros2_ws/
    ROS2 workspace

web_gui/
    MQTT HTML GUI
```
